### Requirements

- Node 6.9.1
- NPM 4.0.2
- PostgreSQL 9.6.2

### Installing Dependencies
From root directory:
```sh
$ npm install
```

### Set up DB and Seed Data
From root directory:
```
$ node MovieData.js
```

### View Movie Data
From postgreSQL:
```
$ SELECT * FROM films;
```
```
$ SELECT * FROM directors;
```
